# ROS 2 Humble Tutorial Playground

This repository contains my completed code up to "Implementing Custom Interfaces" in the [ROS 2 Documentation Humble](https://docs.ros.org/en/humble/Tutorials.html)  
I decided to try and follow the tutorial using a VS Code Dev Container so that I could gain experience using it and avoid some headaches in the future.
I might also implement some things here before my actual project (other parts of the tutorial).

Thanks to the Open Robotics team and all of the great documentation that people created!
